Funciones

Configuracion rapida con controles utiles para uso real

Funciona entre distintos sistemas

Usa un iPhone como camara y un Android como monitor, o al reves. La app esta pensada para entornos con dispositivos mixtos.

Lista para tethering

No hace falta router. Si un dispositivo crea una red compartida, la camara y la pantalla pueden unirse y empezar a transmitir.

Deteccion automatica rapida

El dispositivo de visualizacion puede encontrarse automaticamente en la misma red, reduciendo el tiempo de preparacion.

Conexion por codigo QR

Escanea el codigo QR del dispositivo de visualizacion desde la camara para conectarte sin introducir datos de red manualmente.

Control de retraso intencional

Configura la latencia en el receptor cuando necesites monitorizacion diferida, comparaciones o efectos de presentacion.

Util durante la transmision

El receptor admite zoom, espejo y grabacion. El emisor tambien puede guardar el flujo cuando sea necesario.

FAQ

Preguntas frecuentes

Necesito conexion a internet?

No. No hace falta internet externo siempre que ambos dispositivos puedan verse en la misma red Wi-Fi o tethering.

El QR es la unica forma de conexion?

No. La aplicacion admite tanto deteccion automatica como emparejamiento por QR para elegir la opcion mas rapida.

Puedo usarla entre iOS y Android?

Si. La aplicacion esta disenada para combinaciones multiplataforma donde emisor y receptor usan sistemas moviles distintos.

Se puede grabar el video?

Si. Tanto el emisor como el receptor incluyen opciones para guardar el flujo cuando sea necesario.
