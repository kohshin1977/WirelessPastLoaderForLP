# Wireless Delay Cam LP Public Site

このフォルダは GitHub Pages 用の公開専用ファイル一式です。

## 含まれるもの

- `index.html`
- `styles.css`
- `.nojekyll`
- `assets/`
- `images/`

## 公開手順

1. GitHub で新しい `public` リポジトリを作成する
2. このフォルダの中身をそのリポジトリのルートへコピーする
3. そのリポジトリへ push する
4. GitHub の `Settings` → `Pages` を開く
5. `Source` を `Deploy from a branch` にする
6. ブランチを `main`、フォルダを `/ (root)` にする

## 補足

- Android のストアURLは設定済みです
- iOS の App Store URL は仮URLです
- 画像とバッジはすべて相対パスで参照しています
